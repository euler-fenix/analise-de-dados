# 📊 Relatório de Vendas

Projeto desenvolvido para praticar análise de dados utilizando Excel e Power BI.

## Objetivo

Criar um dashboard para visualizar informações de vendas, lucro, faturamento e desempenho por período.

## Ferramentas utilizadas

- Excel
- Power BI

## Arquivos do projeto

- Base de dados em Excel (vendas)
- Dashboard em Power BI (relatorio-de-vendas)

## Aprendizados

Neste projeto pratiquei organização de dados, criação de indicadores e construção de dashboard visual para análise de vendas.
